## Informações para uso:

```bash
# Clonar o repositório
$ git clone https://github.com/marcosfaneli/face_recognition.git

# Entrar no repositório
$ cd face_recognition

# Instalar dependências
$ pip install opencv-python

# Executar o exemplo
$ python exemplo.py
```
